# Demo database project

Goal of this project is to demonstrate some minor database functionalities, without using an actual database.

The following functionalities are included in the code:
1. Create a new database
2. Create a new table
3. Add new instances to a table
4. Retrieve the data of a table instance
5. Delete existing instances in a table
6. Delete a table
7. Add users to the admin_group
8. Reset the entire database
9. Delete the database

It is possible to execute the code by either making use of the GUI (Gui.py) or the CLI (database.py).
